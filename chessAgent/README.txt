To run a match: 

python3 Play.py --p1 (player1FileName).py --p2 (player2FileName).py 

Example from given files:

python3 Play.py --p1 BreakthroughAI.py --p2 BreakthroughAI.py 
    - BreakthroughAI plays against itself 

python3 Play.py --p1 BreakthroughAI.py --p2 Opponent.py 
    - BreakthroughAI plays against an opponent making random moves
